# QAOA for Max-Cut Optimization

This project explores the Quantum Approximate Optimization Algorithm (QAOA) for solving the Max-Cut problem, a graph-based combinatorial optimization problem.

The goal is to compare QAOA performance across noiseless simulation, noisy simulation, and IBM Quantum hardware using Qiskit. The results are evaluated against classical brute-force solutions using metrics such as cut value, approximation ratio, and runtime.
